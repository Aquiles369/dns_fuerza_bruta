 𓆩⚜𓆪 Técnica de reconocimiento que consiste en probar automáticamente miles de nombres de subdominios usando una wordlist para descubrir cuáles existen en un dominio. Permite identificar activos ocultos (como api, dev, admin o staging) y ampliar la superficie de ataque durante la fase de recon. Es una de las técnicas más utilizadas en Bug Bounty para encontrar objetivos no documentados.<br><br>
